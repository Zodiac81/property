                Bijk 5 months ago

                The user can sign up without passing captcha verification.

                Laravel 5.4

                $data['captcha'] = $this->captchaCheck()

                The expected result is 0 or 1.

                The validation rule always passes $data['captcha']

                'captcha' => 'required|min:1'

                Please change the validation rule to:

                'captcha' => 'accepted'
